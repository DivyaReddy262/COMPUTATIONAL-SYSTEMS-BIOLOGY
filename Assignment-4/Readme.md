# Assignment 4: GSEA and Enrichment Map Analysis

## Overview
This assignment analyzes estrogen treatment expression data from GSE11352 using Gene Set Enrichment Analysis (GSEA) and Enrichment Map visualization in Cytoscape. The goal was to compare treated and untreated samples at 12 hours and 24 hours and identify pathway-level differences.

## Objectives
- Perform GSEA for 12h treated vs untreated samples
- Perform GSEA for 24h treated vs untreated samples
- Visualize enriched pathways using Cytoscape Enrichment Map
- Compare pathway-level differences between 12h and 24h

## Tools Used
- R
- GSEA
- Cytoscape with Enrichment Map

## Files
- code/gsea-analysis.R - R code used for analysis
- report/assignment-4-report.docx - assignment report
- enrichment-map/enrichment-map.cys - Cytoscape session file

## Summary
The analysis compares gene expression responses to estrogen treatment at two time points. GSEA was used to identify enriched pathways, and Cytoscape Enrichment Map was used to visualize pathway relationships and compare biological patterns between 12h and 24h.
